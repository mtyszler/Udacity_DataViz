TITLE: Human Development Report data (1990 - 2015)

# INTRO:
This data set combines data downloaded from the HDR report.
For more details see: http://hdr.undp.org/en/data

and Regional and Development Classifications are from UNSD.
For details see: https://unstats.un.org/unsd/methodology/m49/overview/

Processing of the data is described in Prep_data.html

# DATA Description:
4518 obs in 10 variables

* Country = Factor with 194 levels: Country name
(see list of countries below)

* year = integer 1990 to 2015

* Human Development Index: unitless number from 0-1

* Life Expectancy Index: unitless number from 0-1

* Education Index: unitless number from 0-1

* Income Index: unitless number from 0-1

* Human Development Index Male: unitless number from 0-1

* Human Development Index Female: unitless number from 0-1

* Region = Factor with 5 levels: Region Name
  Africa, Americas, Asia, Europe, Oceania
 
* Development Classification = Factor with 2 levels: Development Classification
  Developing, Developed
